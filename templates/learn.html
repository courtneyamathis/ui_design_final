Boat Types

There are several types of boats, or racing shells.
Read the definition of each key term, then click to see the next.

Oar

The device one uses to pull or push the water, formerly made of wood and now usually made of carbon fiber.

Sculling

The rowing style in which each rower has two oars.

Sweeping

The rowing style in which rowers have one oar each.


Coxswain

The person in the boat responsible for steering, safety, motivating the crew, and race strategy. The coxswain either sits in the stern or lies in the bow of the boat. Pronounced "Cox'n."

How to Read Abbreviations

Each boat type has a different abbreviation made up of two parts, a number and a symbol.
The number represents the number of rowers in the boat. Note this does not include the coxswain.
The symbol represents the style of rowing.
Click on each symbol below to learn which style each represents.

x : sculling style

+ : sweeping style with a coxswain

- : sweeping style without a coxswain
